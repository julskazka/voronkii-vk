# 🚀 Vibe HTML Kit — Стартер для курса вайбкодинга

Это твой **стартовый проект** для создания Vibe-приложений в экосистеме Notibot.  
Просто открой папку в Antigravity IDE и скажи агенту что ты хочешь — он сделает всё сам.

---

## ⚡ Как начать

### 1. Открой папку в Antigravity
Добавь эту папку как workspace в Antigravity IDE.

### 2. Напиши агенту
```
Создай Vibe-лендинг для фитнес-клуба
```
Агент задаст уточняющие вопросы и создаст готовое приложение.

---

## 📦 Структура проекта

```
vibe-html-kit/
├── index.html                        ← твоё Vibe-приложение
├── form_schema_*.json                ← (опционально) схемы форм из Notibot
├── README.md
└── .gemini/
    └── config/
        └── plugins/
            └── vibe-html-kit/
                ├── plugin.json
                └── skills/
                    ├── generate/SKILL.md   ← создать/обновить лендинг
                    ├── github/SKILL.md     ← залить на GitHub
                    └── deploy/SKILL.md     ← опубликовать на Netlify
```

## 🧠 Как создавать новые скилы (Для продвинутых)

Если тебе нужен блок, которого нет в стандартном наборе (например, слайдер картинок или плейлист), ты можешь попросить агента создать под него новый скилл.

Отправь агенту такой запрос:
> *«Используя правила нашего проекта из `skills/rules/SKILL.md`, создай новый скилл для блока **[Название блока, например: Слайдер картинок]** и зарегистрируй его в `plugin.json`. Опиши в SKILL.md триггеры, шаблоны HTML/Tailwind, JS-логику управления и интеграцию с Notibot Bridge (haptics).»*

Агент сам создаст инструкции, запишет их в папку `.gemini/config/plugins/vibe-html-kit/skills/` и зарегистрирует в плагине.
